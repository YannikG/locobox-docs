# Papierkorb

Im Bereich **Papierkorb** findest du geloeschte Eintraege und kannst sie verwalten.

## Typische Aufgaben

- Geloeschte Elemente durchsuchen
- Eintraege wiederherstellen
- Eintraege endgueltig entfernen
