# Nutzung

Im Bereich **Nutzung** ist auf einen Blick sichtbar, wie viele Einträge aktuell genutzt werden und was im aktuellen Plan enthalten ist.

Ohne Premium gelten Limiten pro Bereich. So bleibt sofort sichtbar, wie viel noch frei ist.

Mit Premium gelten diese Limiten nicht. Arbeiten ohne Begrenzung ist möglich.
    
Im gleichen Bereich lässt sich das Premium-Abo direkt starten, verlängern oder anpassen.

Mit **Status aktualisieren** wird der aktuelle Stand neu geladen.
