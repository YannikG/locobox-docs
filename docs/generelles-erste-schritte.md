# Erste Schritte

So startest du mit Locobox:

1. Gehe auf die Hauptseite [www.locobox.ch](https://www.locobox.ch).
2. Klicke auf **Direkt zur App**.
3. Melde dich an, erstelle ein Konto oder waehle **Sign in with Google**.
4. Folge den Anweisungen in der App.
5. Danach landest du auf dem **Dashboard**.

Wenn du auf dem Dashboard bist, kannst du direkt mit der Verwaltung deiner Sammlung starten.
