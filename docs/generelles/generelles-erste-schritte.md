# Erste Schritte

So gelingt der Start mit Locobox:

1. Auf die Hauptseite [www.locobox.ch](https://www.locobox.ch) wechseln.
2. Auf **Direkt zur App** klicken.
3. Anmelden, ein Konto erstellen oder **Sign in with Google** wählen.
4. Den Anweisungen in der App folgen.
5. Danach öffnet sich die [Übersicht](../uebersicht/uebersicht.md).

Auf dem Dashboard kann direkt mit der Verwaltung der Sammlung gestartet werden.
