# Generelles

Wenn Sie [Locobox](https://www.locobox.ch) bereits von der öffentlichen Website kennen, wissen Sie ungefähr, worum es geht: eine **online geführte Modelleisenbahnsammlung**. Locobox hilft Ihnen, Fahrzeuge und Zubehör übersichtlich zu erfassen, in **Kategorien** zu ordnen, **Decoder** zuzuweisen und aus Ihren Modellen **Züge** zusammenzustellen. Die Oberfläche ist auf einen flotten Arbeitsalltag ausgelegt (unter anderem globale Suche, Tabellenansicht, mobiler Zugriff). Alles läuft **im Browser**, ohne Installation auf dem Gerät.

**Schneller Einstieg in die App:** Wer von der Website zur Oberfläche wechseln möchte oder neu einsteigt, folgt den konkreten Schritten unter [Erste Schritte](./generelles-erste-schritte.md).

**Technisches Grundsetup:** Welche Browser zuverlässig unterstützt sind und woran es liegt, wenn etwas seltsam wirkt, steht unter [Welcher Browser für Locobox?](./generelles-browser-kompatibilitaet.md).

**Suche überall auf einen Schlag:** Die zentrale Suche über Artikel, Kategorien, Decoder und Züge ist unter [Globale Suche](./globale-suche.md) beschrieben.
