# Locobox Dokumentation

Diese Website basiert auf [Docusaurus](https://docusaurus.io/), einem modernen Static-Site-Generator.

## Installation

```bash
npm install
```

## Lokale Entwicklung

```bash
npm run start
```

Der Befehl startet einen lokalen Dev-Server und öffnet ein Browserfenster. Die meisten Änderungen sind sofort sichtbar, ohne Neustart.

## Build

```bash
npm run build
```

Der Befehl erzeugt statische Inhalte im Ordner `dist`. Dieser Output kann von deinem Hosting direkt veröffentlicht werden.

## Deployment

Für eine lokale Vorschau des produktiven Outputs:

```bash
npm run serve
```
